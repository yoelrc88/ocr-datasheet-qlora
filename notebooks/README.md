# Notebooks

Do not hand-edit `olmocr_table_html_colab.ipynb` directly.

It is generated by:

- `scripts/generate_colab_notebook.py`

Regenerate it with:

```bash
python3 scripts/generate_colab_notebook.py
```

Why this is the safer approach:

- raw `.ipynb` JSON is easy to break by hand
- generated notebooks are reproducible
- code review is easier when the source-of-truth is a normal script
- Colab output cells should not be committed as the notebook source-of-truth
